# GEOFlow — Product Context

## Overview
GEOFlow is an open-source Generative Engine Optimization (GEO) content engineering and multi-site distribution system.

## Core Purpose
An AI-powered content factory that:
1. Ingests knowledge (PDFs, URLs, text)
2. Generates articles using AI (OpenAI, Gemini, DeepSeek)
3. Reviews and publishes articles
4. Distributes content to multiple sites (WordPress, HTTP API, GEOFlow Agent)
5. Tracks AI crawler analytics

## Target Users
- SEO professionals and content marketers
- Website operators managing multiple sites
- AI-focused content teams

## Key Differentiators
- **AI-native CMS**: Not a traditional CMS — built around AI generation workflows
- **RAG integration**: Knowledge Base → AI generation pipeline with pgvector
- **Multi-site distribution**: Publish once, distribute to many sites automatically
- **GEO focus**: Optimized for AI search engines (ChatGPT Search, Perplexity), not just Google
